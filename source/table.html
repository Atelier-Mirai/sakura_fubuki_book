セレクター名 選択するもの 例
要素セレクタ　　　　　 指定された型のすべての HTML 要素 p
<p> を選択
ID セレクタ 指定された ID を持つページ上の要素です。
指定された HTML ページでは、各 id 値は
一意でなければなりません。		 #my-id
<p id="my-id"> または
<a id="my-id"> を選択
クラスセレクタ 指定されたクラスを持つページ上の要素です。
同じクラスを持つ複数の要素が 一つのページに現れることがあります。		 .my-class
<p class="my-class"> および <a class="my-class"> を選択
